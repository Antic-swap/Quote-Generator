# Quote Generator

## 📌 Description

This is a simple Quote Generator web application that displays a random quote along with its author when the user clicks a button. The project is built using HTML, CSS, and JavaScript to practice core frontend concepts.

---

## 🚀 Features

* Generate random quotes on button click
* Displays both quote and author
* Clean and simple UI
* Fast and lightweight

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

---

## 🧠 What I Learned

* DOM manipulation (`querySelector`)
* Event handling (`addEventListener`)
* Working with arrays and objects
* Generating random values using `Math.random()`

---

## 📂 How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
